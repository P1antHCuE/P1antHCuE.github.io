    从0开始学习渗透

    P1ant731
    2025-04-22

    让ai帮忙给了一份学习路线看看我能坚持多久，第一个月的内容比较基础，跳过一下，从第二个月开始。

    今天是2025年4月22日周二，从明天23号开始我将全程记录我的学习过程。

    📅 第2个月：信息安全基础 & 工具入门（详细版）

    目标：掌握渗透测试基本工具（Nmap、Burp Suite、Metasploit），理解漏洞原理。

    📌 第5周：信息安全基础 & 环境搭建

    日期 任务
    周一 ✅ 学习CIA三元组（机密性、完整性、可用性），了解OWASP Top 10。
    周二 ✅ 安装Kali Linux（虚拟机或物理机），配置基础环境（更新apt）。
    周三 ✅ 搭建DVWA（Damn Vulnerable Web App）靶机，熟悉其界面。
    周四 ✅ 学习HTTP协议，用浏览器开发者工具分析请求（查看Headers、Cookies）。
    周五 ✅ 使用curl和Wget手动发送HTTP请求，理解GET/POST区别。
    周六 🔧 实战：修改DVWA的难度级别，尝试触发一个XSS弹窗（<script>alert(1)</script>）。
    周日 📝 整理笔记：记录信息安全核心概念、HTTP协议要点。

    📚 推荐资源

    • DVWA安装：DVWA GitHub
    • HTTP协议：MDN HTTP教程

    📌 第6周：信息收集 & 扫描（Nmap、Whois）

    日期 任务
    周一 ✅ 学习Nmap基础：nmap -sV <IP>（扫描开放端口和服务版本）。
    周二 ✅ 使用Nmap脚本（--script参数），如vulners扫描漏洞。
    周三 ✅ 学习Whois查询域名信息，theHarvester收集邮箱/子域名。
    周四 ✅ 使用Shodan（搜索引擎）查找暴露的物联网设备。
    周五 ✅ 实战：扫描自己的路由器（nmap 192.168.1.1），分析结果。
    周六 🔧 挑战TryHackMe房间：Nmap。
    周日 📝 整理笔记：Nmap常用命令、信息收集方法论。

    📚 推荐资源

    • Nmap备忘单：Nmap Cheat Sheet
    • Shodan：Shodan搜索技巧

    📌 第7周：Web安全入门（Burp Suite、SQL注入）

    日期 任务
    周一 ✅ 安装Burp Suite Community，配置浏览器代理（127.0.0.1:8080）。
    周二 ✅ 拦截DVWA登录请求，修改参数（如admin' OR '1'='1尝试SQL注入）。
    周三 ✅ 学习SQL注入原理，手动利用UNION SELECT提取数据库信息。
    周四 ✅ 使用SQLmap自动化测试：sqlmap -u "http://target.com?id=1" --dbs。
    周五 ✅ 实战：在DVWA上完成Low/Medium难度的SQL注入挑战。
    周六 🔧 尝试PortSwigger Lab：SQL注入。
    周日 📝 整理笔记：Burp Suite拦截、SQL注入Payload集合。

    📚 推荐资源

    • Burp Suite教程：PortSwigger Academy
    • SQL注入Payload：PayloadsAllTheThings

    📌 第8周：Metasploit框架 & 漏洞利用

    日期 任务
    周一 ✅ 学习Metasploit基础：msfconsole，search漏洞模块。
    周二 ✅ 使用exploit/multi/handler监听反弹Shell（结合msfvenom生成Payload）。
    周三 ✅ 攻击Metasploitable靶机（如exploit/unix/ftp/vsftpd_234_backdoor）。
    周四 ✅ 学习后渗透命令（meterpreter的getsystem、hashdump）。
    周五 ✅ 实战：在TryHackMe完成Metasploit房间。
    周六 🔧 挑战HTB“Starting Point”机器（如Meow）。
    周日 📝 整理笔记：Metasploit常用命令、漏洞利用流程。

    📚 推荐资源

    • Metasploit指南：Metasploit Unleashed
    • HTB Starting Point：Hack The Box

    📅 第3个月：Web安全专项（超详细版）

    目标：深入理解OWASP Top 10漏洞，掌握Burp Suite高级功能。

    📌 第9周：SQL注入进阶

    日期 任务
    周一 ✅ 学习盲注（Boolean-Based/Time-Based），手工构造Payload。
    周二 ✅ 使用SQLmap的--level和--risk参数提高检测深度。
    周三 ✅ 绕过WAF（如/**/代替空格，CHAR()编码）。
    周四 ✅ 实战：在PortSwigger Lab完成SQL盲注。
    周五 ✅ 学习Out-of-Band（OOB）注入，利用DNS外带数据。
    周六 🔧 挑战DVWA的Impossible级别SQL注入（理解防御措施）。
    周日 📝 整理笔记：SQL注入绕过技巧、自动化工具对比。

    📌 第10周：XSS & CSRF

    日期 任务
    周一 ✅ 学习反射型XSS，在DVWA中弹窗（<script>alert(document.cookie)</script>）。
    周二 ✅ 存储型XSS：在留言板注入持久化脚本。
    周三 ✅ DOM型XSS：分析前端JS代码（如eval()漏洞）。
    周四 ✅ 使用BeEF框架（Browser Exploitation）劫持会话。
    周五 ✅ CSRF攻击：伪造请求修改DVWA用户密码（Burp Suite生成PoC）。
    周六 🔧 实战：PortSwigger的XSS Labs。
    周日 📝 整理笔记：XSS类型对比、CSRF防御方法（SameSite Cookie）。

    📌 第11周：文件上传漏洞 & SSRF

    日期 任务
    周一 ✅ 学习文件上传漏洞原理，理解MIME类型绕过、黑名单绕过。
    周二 ✅ 实战：在DVWA中上传Webshell，尝试绕过文件类型限制。
    周三 ✅ 学习SSRF（服务器端请求伪造）原理，构造内网探测Payload。
    周四 ✅ 使用Burp Suite Intruder测试SSRF，尝试访问本地文件（如file:///etc/passwd）。
    周五 ✅ 实战：在PortSwigger Lab完成SSRF挑战。
    周六 🔧 挑战TryHackMe房间：SSRF。
    周日 📝 整理笔记：文件上传漏洞防御措施、SSRF利用技巧。

    📚 推荐资源

    • 文件上传漏洞：OWASP文件上传安全
    • SSRF教程：SSRF指南

    📌 第12周：命令注入 & XXE

    日期 任务
    周一 ✅ 学习命令注入原理，理解&&、`
    周二 ✅ 实战：在DVWA中利用命令注入执行id、whoami等命令。
    周三 ✅ 学习XML外部实体注入（XXE）原理，构造读取本地文件的Payload。
    周四 ✅ 使用Burp Suite测试XXE，尝试读取敏感文件（如/etc/passwd）。
    周五 ✅ 实战：在PortSwigger Lab完成XXE挑战。
    周六 🔧 挑战TryHackMe房间：XXE。
    周日 📝 整理笔记：命令注入防御方法、XXE利用与防御技巧。

    📚 推荐资源

    • 命令注入：PayloadsAllTheThings
    • XXE教程：OWASP XXE

    📅 第4个月：系统 & 网络渗透（每日计划）

    🎯 目标

    1. 掌握 Windows/Linux提权技术（内核漏洞、配置错误）。
    2. 学会 密码破解（Hashcat、John the Ripper）。
    3. 理解 Wi-Fi安全（WPA2破解、中间人攻击）。
    4. 入门 内网渗透（横向移动、权限维持）。

    📌 第13周：Windows/Linux提权

    核心知识点：内核漏洞、SUID/SGID、服务滥用、令牌窃取。

    日期 任务 具体操作
    周一 ✅ Windows提权基础 - 学习whoami /priv查看权限。 - 使用Windows-Exploit-Suggester找漏洞。
    周二 ✅ 内核漏洞提权（如PrintNightmare） - 复现CVE-2021-1675： python3 CVE-2021-1675.py -u user -p pass -d IP。
    周三 ✅ Linux提权：SUID/SGID - 查找SUID文件：find / -perm -4000 2>/dev/null。 - 利用/bin/bash -p提权。
    周四 ✅ Cron Jobs提权 - 检查可写的Cron任务：cat /etc/crontab。 - 写入反向Shell脚本。
    周五 ✅ Docker逃逸 - 检查docker.sock挂载：ls -la /var/run/docker.sock。 - 尝试chroot逃逸。
    周六 🔧 实战：HTB靶机（如Jerry） - 利用Tomcat WAR部署获取Shell，提权至NT AUTHORITY\SYSTEM。
    周日 📝 整理笔记 - 记录Windows/Linux提权命令和漏洞复现步骤。

    📚 推荐资源

    • 提权命令备忘单：GTFOBins（Linux）、LOLBAS（Windows）。
    • HTB靶机：TJ Null’s OSCP List。

    📌 第14周：密码破解 & Wi-Fi安全

    核心知识点：哈希破解、WPA2握手包捕获、中间人攻击。

    日期 任务 具体操作
    周一 ✅ 密码破解基础 - 学习哈希类型识别：hash-identifier。 - 使用John破解Linux shadow文件： john --wordlist=rockyou.txt hashes.txt。
    周二 ✅ Hashcat高级用法 - 破解NTLM哈希： hashcat -m 1000 hashes.txt rockyou.txt。 - 使用规则攻击（-r参数）。
    周三 ✅ Wi-Fi嗅探（Aircrack-ng） - 捕获握手包： airodump-ng -c 6 --bssid AP_MAC -w capture wlan0mon。
    周四 ✅ WPA2破解 - 使用aircrack-ng： aircrack-ng -w rockyou.txt capture.cap。
    周五 ✅ 中间人攻击（MITM） - 使用ettercap毒化ARP： ettercap -T -i eth0 -M arp /target_IP// /gateway_IP//。
    周六 🔧 实战：破解Wi-Fi（需授权） - 在虚拟机中模拟攻击（如用hostapd创建测试AP）。
    周日 📝 整理笔记 - 记录Hashcat模式、Wi-Fi破解流程。

    📚 推荐资源

    • Hashcat模式列表：Hashcat示例。
    • Wi-Fi安全实验室：WiFi Pineapple。

    📌 第15周：内网渗透 & 横向移动

    核心知识点：NTLM Relay、Pass the Hash、BloodHound。

    日期 任务 具体操作
    周一 ✅ 内网信息收集 - 使用nmap扫描内网： nmap -sn 192.168.1.0/24。 - 查找SMB共享：smbclient -L //IP。
    周二 ✅ Pass the Hash（PtH） - 使用pth-winexe： pth-winexe -U admin%NTLM_HASH //IP cmd。
    周三 ✅ NTLM Relay攻击 - 使用Responder+Impacket： ntlmrelayx.py -tf targets.txt -smb2support。
    周四 ✅ BloodHound分析域环境 - 上传Sharphound数据： bloodhound-python -d domain -u user -p pass -ns IP。 - 分析路径。
    周五 ✅ 权限维持（Golden Ticket） - 使用mimikatz伪造票据： kerberos::golden /user:admin /domain:test.com /sid:S-1-5-21-...。
    周六 🔧 实战：HTB靶机（如Active） - 利用SMB漏洞获取Shell，横向移动至域控。
    周日 📝 整理笔记 - 记录内网渗透命令、BloodHound使用技巧。

    📚 推荐资源

    • Impacket示例：Impacket指南。
    • BloodHound教程：BloodHound官方文档。

    📌 第16周：综合实战 & 挑战HTB中级靶机

    日期 任务 靶机推荐
    周一 🔧 HTB靶机：Legacy - 利用SMB漏洞（MS08-067）提权。
    周二 🔧 HTB靶机：Blue - 利用EternalBlue（MS17-010）获取SYSTEM权限。
    周三 🔧 HTB靶机：Devel - 通过FTP弱口令上传Webshell，提权。
    周四 🔧 HTB靶机：Optimum - 使用HFS漏洞获取初始立足点，内核提权。
    周五 🔧 HTB靶机：Bastion - 利用备份文件泄露获取密码，SSH登录提权。
    周六 📝 复盘总结 - 记录每台靶机的漏洞利用链和提权方法。
    周日 🎯 自由挑战 - 尝试TryHackMe的内网渗透房间。
